# Sasol_Stock_Prediction
Attempt at predicting a next day stock movements for Sosol Ltd (ticker = SASOL SJ Equity). Data is acquired through Yahoo finance API.

## Variables:
- Closing price
- Volume

## Predictive Model
- Conditional Random Field
